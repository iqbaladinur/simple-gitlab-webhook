# simple-gitlab-webhook
Simple gitlab webhooks event notification using telegram bot.

## Installation
`composer install`

## Config
`GITLAB_TOKEN`

`$TOKEN  = "YOUR_TELEGRAM_BOT_TOKEN"`

`$chatid = "CHAT_ID"`

## Test & Run Local
`php -S localhost:8080`

See https://docs.gitlab.com/ee/user/project/integrations/webhooks.html
